## 목적
- React의 전반적인 사용법을 익힌다.
- REST API를 경험해 보기.  
- Redux를 사용해 상태를 관리해 보기.  
- Axios로 비동기 네트워크 요청해 보고 요청 결과를 다루어 화면에 렌더링 하기.
