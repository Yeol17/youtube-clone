## 프로젝트 목적
1. 리액트 사용 경험과 숙련도 향상
2. Redux를 이용한 상태관리를 경험
3. CSS 전처리기 SCSS 의 사용 경험
4. Axios를 사용해 REST API 호출을 해보고 응답받은 리소스를 다루는 경험
